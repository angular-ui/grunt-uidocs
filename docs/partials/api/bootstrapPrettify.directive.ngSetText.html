ngSetText
directive in module bootstrapPrettify

Description

A directive that sets Text in a way that is supported by IE8.

Dependencies

  • getEmbeddedTemplate

Usage

as attribute
<ANY ng-set-text>
   ...
</ANY>
as class
<ANY class="ng-set-text">
   ...
</ANY>

Directive info

  • This directive executes at priority level 10.
