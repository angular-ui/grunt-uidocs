popoverElement
service in module bootstrap

Description

Provides helper functions for the popover directive

Methods

  • content(value)

    Runs the value passed in through the marked library and updates the value of the content of the popover element.

    Parameters
    ParamTypeDetails
    valuestring

    New content for the popover element.

    Returns
    string

    The new value of the content of the popover element.

  • hide()

    Removes visible class to the popover element on the page and set its position to be out of bounds.

  • init()

    Attaches a popover element to the body of the page.

  • isSituatedAt(element)

    Returns whether or not the popover element is situated next to the element passed in.

    Parameters
    ParamTypeDetails
    elementDOMElement

    Element to check against.

    Returns
    boolean

    Returns true is the current element is the besideElement

  • position(x, y)

    Sets the top and left position of the popover element if x and y are not null. Otherwise returns the x and y coordinates of the popover element.

    Parameters
    ParamTypeDetails
    xnumber

    Left position for the popover element.

    ynumber

    Top position for the popover element.

    Returns
    object | null

    The current x and y coordinates of the popover element or nothing if valid x and y coordinates are provided.

  • positionArrow(position)

    Adds the position class for the arrow on the popover element.

    Parameters
    ParamTypeDetails
    positionstring

    Position class for popover arrow.

  • positionAway()

    Sets besideElement to null and hides the popover element.

  • positionBeside(element)

    Sets besideElement to element passed in and shows the popover element.

    Parameters
    ParamTypeDetails
    elementDOMElement

    Element to position the popover besides.

  • show()

    Adds visible class to the popover element on the page and set its position.

  • title(value)

    Updates the title value of the popover element.

    Parameters
    ParamTypeDetails
    valuestring

    New title for the popover element.

    Returns
    string

    The new value of the title of the popover element.

  • visible()

    Returns whether or not the popover element is visible.

    Returns
    boolean

    true if the elements y position is greater than 0.
