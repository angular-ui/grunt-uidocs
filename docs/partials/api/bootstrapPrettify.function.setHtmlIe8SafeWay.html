setHtmlIe8SafeWay
function in module bootstrapPrettify

Description

Ensures that pre tag does not lose line breaks on IE8.

  • http://stackoverflow.com/questions/451486/pre-tag-loses-line-breaks-when-setting-innerhtml-in-ie
  • http://stackoverflow.com/questions/195363/inserting-a-newline-into-a-pre-tag-ie-javascript

Usage

new setHtmlIe8SafeWay(element, html);

Parameters

ParamTypeDetails
elementDOMElement

DOM Element to be updated

htmlstring

HTML to be added inside the pre tag

Returns

DOMElement

The element with its contents updated.
