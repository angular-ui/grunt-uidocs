ngSetHtml
directive in module bootstrapPrettify

Description

A directive that sets HTML in a way that is supported by IE8.

Dependencies

  • getEmbeddedTemplate

Usage

as attribute
<ANY ng-set-html>
   ...
</ANY>
as class
<ANY class="ng-set-html">
   ...
</ANY>

Directive info

  • This directive executes at priority level 10.
